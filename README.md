# rcti
Interactive CLI for react 
```jaascript
npm install -g rcti
```
## create new Project
```javascript
rcti new <project-name> 
```
## create component
```javascript
rcti c <path(optional)/component-name>
```
## create  page
```javascript
rcti p <path(optional)/page-name>
```
### create service 
```javascript
rcti s <path(optional)/service-name>
```
### serve project
```javascript
rcti serve
```
### build project
```javascript
rcti build
```
